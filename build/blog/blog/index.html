1.概念引入#

  • HTML

    HTML (HyperText Markup Language) 不是一门编程语言，而是一种用来告知浏览器如何组织页面的标记语言

    HTML（超文本标记语言） 是一种描述语言，用来定义网页结构

  • HTML文档

    HTML 文档是包含多个 HTML 元素 的文本文档。每个元素都用一对开始和结束 标签 包裹。每个标签以尖括号（< >）开始和结束。也有一部分标签中不需要包含文本，这些标签称为空标签(自结束标签)，如 <img>
